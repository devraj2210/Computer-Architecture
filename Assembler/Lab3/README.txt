The "input.s" file must have the following properties: 1. the words in the instructions must be separated by spaces or commas
                                                       2. there must not be any empty lines
                                                       3. there must only be one instruction in each line
                                                       
Instructions to run the program: 1. type "make" in the terminal and run
                                 2. type "./riscv_asm" in the terminal and run

A message saying "Successfully Assembled!" will be printed if finished execution without error, an error message is printed otherwise.
