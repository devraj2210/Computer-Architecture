The input files must have the following properties: 1. the words in the instructions must be separated by spaces or commas
                                                       2. there must not be any empty lines
                                                       3. there must only be one instruction in each line
                                                       4. If data is initialized with .data, text must also be initialized with .text
                                                       
Instructions to run the program: 1. type "make" in the terminal and run
                                 2. type "./riscv_sim" in the terminal and run

The programs run indefinitely until an "exit" command is given.
